Property based testing

Van specifieke scenario's naar generieke properties

Anno 2024 zijn de meeste software developers wel bekend met het schrijven van unit en systeem tests. In deze vorm van testen word uitgegegaan van een specifiek scenario waarna een of meerdere eigenschappen van het lijdend voorwerp worden getest. In dit vorbeeld gaan we uit van de stack ADT.


interface Stack {
    T pop();

    T top();

    void push(T elem);
}

Een van de unit test voor een implementatie van deze stack zal er ongeveer zo uit zien:
